Frequently Used Effects

EZmenu recognises the usual HTML tags (e.g. <b>, <img>, <br>) in the panel entries, and these tags can be used to display images, multiple lines and different font styles in the menus. See this page for examples.

Note:

1. You need to terminate the font style (e.g. with </b>) or the rest of the menu would be affected.

2. Use only SINGLE quotes in the panel entries, e.g <img src='..'>.

3. It is a good idea to specify the width and height of any image, as it makes it easier and faster for EZmenu to format the menus.
