Centering Panels

Instead of using showPermPanel to display permanent panels, you may use the command showPermPanelCentered. Panels displayed using this command will be centered around the x and y coordinates specified.

The format is :

<script> showPermPanelCentered('Panel Name', x-coordinate, y-coordinate) </script>

Panel Name = Name of the panel to be displayed (This is the name as entered in the Menu Maker). Remember to enclose the name within single quotes.

x-coordinate, y-coordinate = The location of the point on the webpage where the center point of the panel will be placed.
