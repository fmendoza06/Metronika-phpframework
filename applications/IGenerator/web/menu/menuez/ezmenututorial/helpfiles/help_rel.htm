Relative Positioning

Relative Positioning is achieved by using the EZmenu commands "showRelativePanel" in place of "showPanel". The panel is shown at the point where the mouse first moves (or is clicked) over the invoking image or text.

The format of the command showPanel is:

showRelativePanel('Panel Name', event)

Example: Mouse Over Hyperlink
Move Mouse Over Here
The html code for the above is:

<a href=# onmouseover="showRelativePanel('Mag',event)" onmouseout="hidePanel()">Move Mouse Over Here</a>

Explanation

The above code tells the browser to display the menu Panel "Mag" at the position where the mouse first moves over the text between the <a> and </a> tags and to hide this panel when the mouse is moved out of it.

Example: Mouse Over Image
The html code for the above is:

<a onmouseover="showRelativePanel('Search',event)" onmouseout="hidePanel()"><img name="icon" src="../images/searchhi.gif" border=0></a>
