Display Menus from Images/Text/Hyperlinks

To display a menu panel when the mouse is moved over or clicked on an image or text/hyperlink, use the EZmenu commands "showPanel" and "hidePanel" together with Javascript event handlers "onmouseover", "onclick" and "onmouseout".

The format of the command showPanel is:

<script> showPanel('Panel Name', x-coordinate, y-coordinate) </script>

Example: Mouse Over Hyperlink
Move Mouse Over Here
The html code for the above is:

<a href=# onmouseover="showPanel('Mag',180,250)" onmouseout="hidePanel()">Move Mouse Over Here</a>

Explanation

The above code tells the browser to display the menu Panel "Mag" at position (180,250) when the mouse moves over the text between the <a> and </a> tags and to hide this panel when the mouse is moved out of it.

Example: Mouse Over Image
The html code for the above is:

<a onmouseover="showPanel('Search',100,510)" onmouseout="hidePanel()"><img name="icon" src="../images/searchhi.gif" border=0></a>

The difference between showPanel and showPermPanel is that a panel displayed using the former does not stay permanently on the webpage and will disappear after the mouse is moved out of it.
