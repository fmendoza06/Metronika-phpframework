Design Menu Panels with the Menu Maker

What is a Menu Panel?

A menu system is made up of individual menu panels linked together. A menu panel may be horizontal or vertical as illustrated below.

Using the Menu Maker to generate Menu Panels

The Menu Maker is designed to allow you to create your menu panels easily and quickly. The few steps involved in using the Menu Maker to create menus are described here in detail. We will be using the menu on this page as an example.

Step 1

Ok, let's start! In order to create a menu, you must first specify how the menu will look. This includes setting the colors, the font styles, etc, of the menu panels. The "Panel Properties" box in the Menu Maker allows you to do just that. (For a detailed description of the various properties, click here.)

Step 2

Once you have determined the panel appearance, click the "Generate Header" button. This will add a line of text to the "Panel Entries" Box. This line contains all the formatting information that you have entered above, and will govern the appearance of the panels that are defined below it (See Step 3). You can have more than one header, as the panels will take the properties of the nearest header above it.

Step 3

Next you have to define your menu panels after the header. First, let's explore the example menu. You will find that there are five menu panels altogether (the main bar, the "Magazines" submenu, the "Computers" submenu, the "Science" submenu and the "Search Engine" submenu).

To create the menu, we have to key in the contents of each panel after the menu header in the "Menu Entries" box as follows:

Panel Entries

Type:Bar,.....

Panel=Bar
!<b>Web-NOVA</b>^http://web-nova.com
Magazines^^Mag
Search Engines^^Search

Type:Vertical,.....

Panel=Mag
Computers^^Comp
Science^^Sci

Panel=Search
Yahoo^http://yahoo.com
Lycos^http://lycos.com

Panel=Comp
PC Magazine^http://pcmag.com
PC World^http://pcworld.com

Panel=Sci
Scientific America^http://sciam.com
New Scientist^http://newscientist.com

The keyword "Panel=" signifies the start of a new panel (Important: There is no space before or after the "="). This is followed by the name of the panel. You may give a panel any name you like.

What follows are the individual menu items (Each new line is considered a separate entry). The format is :

"Item Name^Url,target^Child Panel"

To specify an item as a title item, prefix the item name with a "!" (as in !Web-NOVA).

The target is the name of the frame/window that you want the link to open in (If target is not specified, the link opens in the same frame/window). If the item links to a submenu, the name of the panel to link to is specified as shown above.

Step 4

We are almost there! Now, click on the "Generate Code" button. This will translate what you have keyed in above into Javascript code in the "Menu Javascript Code" box in the Menu Maker.

Step 5

Now click the "Highlight Code" button to highlight the generated Javascript code. Copy and paste it in a text editor and save it as "ezdata.js" (or any name *.js).
